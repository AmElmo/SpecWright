# Dismiss suggestions

**Issue ID**: ENG-004  
**Status**: pending  
**Category**: fullstack  
**Complexity Score**: 3/10  
**Estimated Hours**: 2h  
**Complexity Reasoning**: Simple UI interaction with basic API call - straightforward implementation.

## Description

Add the ability to dismiss suggestions that aren't relevant. Dismissed suggestions are hidden from the list and tracked for improving AI accuracy over time.

## Technical Details

- Dismiss button with optional feedback dropdown
- API endpoint to record dismissal with reason
- Analytics logging for AI improvement feedback loop
- Smooth fade-out animation on dismissal

## Dependencies

- ENG-002 (view suggestions) - need suggestion cards to add dismiss button to

## Acceptance Criteria

- [ ] "Dismiss" button visible on all suggestion cards
- [ ] Clicking shows optional feedback dropdown
- [ ] Can dismiss without providing feedback
- [ ] Dismissed suggestion removed from list with animation
- [ ] Category count decreases
- [ ] Cannot undo dismissal (intentional)
- [ ] Dismissal logged to analytics

## Test Strategy

### Automated Tests

- Unit test: Dismiss action removes item from list
- Unit test: Category count updates correctly
- API test: Dismissal recorded in database

### Manual Verification (Human-in-the-Loop)

1. Expand a suggestion card
2. Click "Dismiss" and optionally select a reason
3. Verify suggestion fades out and count updates
4. Refresh page and verify suggestion stays dismissed
