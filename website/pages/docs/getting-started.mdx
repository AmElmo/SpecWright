# Getting Started

Get up and running with SpecWright in under a minute.

## Quick Start

The fastest way to try SpecWright:

```bash
npx specwright
```

This launches the interactive CLI, which will guide you through creating your first specification.

## Installation

For regular use, install globally:

```bash
npm install -g specwright
```

Then run from any directory:

```bash
specwright
```

## Using the Web UI

SpecWright includes a built-in web interface for a more visual workflow:

```bash
specwright --ui
```

This opens the web UI in your browser, where you can manage specs, review phases, and track progress.

## Project Structure

After running SpecWright, your outputs are saved in the `outputs/` directory:

```
outputs/
├── pm-spec.md          # Product requirements
├── design-spec.md      # Design specifications
├── engineering-spec.md # Technical specifications
├── breakdown.md        # Task breakdown
└── status.json         # Project state
```
