# CLI Reference

## Basic Usage

```bash
specwright [options]
```

## Options

| Option | Description |
| --- | --- |
| `--ui` | Launch the web UI instead of the CLI |
| `--version` | Show the installed version |
| `--help` | Display help information |

## Development Mode

When working on SpecWright itself, use the dev command:

```bash
specwright-dev [options]
```

This runs with debug logging enabled for development and troubleshooting.

## Environment Variables

| Variable | Description |
| --- | --- |
| `SPECWRIGHT_DEBUG` | Enable debug logging |
| `SPECWRIGHT_TRACE` | Enable trace-level logging |
