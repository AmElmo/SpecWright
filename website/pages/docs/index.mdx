# Introduction

SpecWright is an AI-powered specification engine for **Specification-Driven Development (SDD)**.

It transforms a simple idea into structured, actionable specs — covering product requirements, design, engineering breakdown, and task planning — so you can feed them directly into your AI coding tools and ship with confidence.

## Why SpecWright?

Most AI-assisted development starts with a vague prompt. SpecWright ensures you start with a **structured specification** instead:

- **PM Phase** — Define requirements, user stories, and acceptance criteria
- **Designer Phase** — Generate UI/UX specifications and component breakdowns
- **Engineer Phase** — Produce technical specs, architecture decisions, and implementation plans
- **Breakdown Phase** — Split work into actionable, scope-appropriate tasks

## How it works

1. Run `npx specwright` or use the web UI
2. Describe your idea in plain language
3. SpecWright walks you through each phase with AI assistance
4. Get structured output files ready for your team or AI editor
